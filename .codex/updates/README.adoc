=== 详细描述

- 在“任务类型”章节中新增以下内容:

- 增加“任务类型选择指引”小节:
** 说明当需求只要求“列清单/拆分任务”时必须使用 PLAN。

- 增加“任务类型与状态联动”说明:
** PLAN 任务完成后产出子任务，子任务可作为 FUNC 任务执行。
** 禁止在 PLAN 任务中直接标记 DONE 且无任务清单输出。
