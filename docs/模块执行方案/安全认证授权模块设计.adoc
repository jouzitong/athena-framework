= Athena Framework 安全认证授权模块设计
:toc: left
:toclevels: 3
:sectnums:

== 1. 文档说明

=== 1.1 目标

.构建一个可插拔的 Spring Boot 安全框架，支持：
* 用户认证（登录、token）
* 权限控制（接口 / 方法）
* 菜单控制（前端展示）
* 角色（可选）
* 数据权限（预留）
* 审计日志

=== 1.2 设计原则

* 可插拔（starter 方式接入）
* 模块解耦（按能力拆分）
* 角色非必选
* 权限统一以 permissionCode 为核心
* 支持覆盖默认实现
* 支持渐进式扩展

== 3. 核心设计

=== 3.1 统一认证模型

[source,java]
----
public class LoginUser {
private Long userId;
private String username;
private String tenantId;
private Set<String> permissions;
}
----

=== 3.2 核心接口定义

==== 用户加载

[source,java]
----
public interface AuthUserService {
LoginUser loadByUsername(String username);
LoginUser loadByUserId(Long userId);
}
----

==== 权限服务

[source,java]
----
public interface PermissionService {
    Set<String> getPermissions(Long userId);
}
----

==== 菜单服务

[source,java]
----
public interface MenuService {
    List<MenuNode> getMenus(Long userId);
}
----

==== Token 服务

[source,java]
----
public interface TokenService {
    String createToken(LoginUser user);
    LoginUser parseToken(String token);
    void invalidate(String token);
}
----

==== 数据权限（预留）

[source,java]
----
public interface DataPermissionService {
    DataScope getDataScope(Long userId, String permission);
}
----

== 4. 认证设计

=== 4.1 登录流程

[source,text]
----
用户请求登录
↓
AuthUserService 查询用户
↓
校验密码
↓
生成 LoginUser
↓
TokenService 生成 token
↓
返回 token
----

=== 4.2 Token 方案

支持两种模式：

[cols="1,2"]
|===
|类型 |说明

|JWT
|无状态，简单

|Redis Token
|支持踢下线、实时失效
|===

配置：

[source,yaml]
----
auth:
  token-type: jwt
----

---

== 5. 授权设计

=== 5.1 权限模型

统一使用：

[source,text]
----
permissionCode

示例：

* user:create
* user:update
* order:list
----

=== 5.2 鉴权方式

==== URL 鉴权

[source,java]
----
/api/user/** -> user:read
----

==== 方法鉴权

[source,java]
----
@RequirePermission("user:create")
----

或：

[source,java]
----
@PreAuthorize("hasAuthority('user:create')")
----

---

== 8. 自动装配设计

=== 8.1 自动配置类

[source,java]
----
@Configuration
@ConditionalOnProperty(prefix="auth", name="enabled", havingValue="true", matchIfMissing=true)
public class SecurityAutoConfiguration {
}
----

=== 8.2 默认 Bean 策略

[source,text]
----
提供默认实现
↓
业务可覆盖
----

示例：

[source,java]
----
@ConditionalOnMissingBean
@Bean
public TokenService tokenService() {
    return new JwtTokenService();
}
----

== 10. 配置设计

[source,yaml]
----
auth:
  enabled: true
  token-type: jwt
  token-header: Authorization
  token-prefix: Bearer
  access-token-expire-minutes: 120
  ignore-urls:

----
